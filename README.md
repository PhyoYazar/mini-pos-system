# mini-pos-system

## To run react "client"

1. cd mini-pos-system
2. cd client
3. yarn
4. add .env file from example.env file
5. yarn dev

## To run nodejs "api"

1. cd mini-pos-system
2. cd api
3. yarn
4. add .env file from example.env file
5. yarn dev
